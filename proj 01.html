Chess Feild Breakdown

this is a personal project that aims to for a realistic scene along with epic proportions to create a feelling of awe and gradure which i think it has achived satisfactorly

for the modeling of the chess peices, ground, cloth and other objects i used blender. and for texturing the entire scene i exported it into substance painter. then back to blender to set up the scene and lighting after which i render the animation using different render layers to breakup the scene so that redering wont take forever . finally i imported the renders into davinci resolve to comple the rendered layers and twick some things for the finall compositing
